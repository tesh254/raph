Install, index, connect an agent.

raph config init
export OPENROUTER_API_KEY="your-key"
raph init --path .
raph agents mcp setup --path .
raph start

Setup flow

The usual local workflow.

1. Bootstrap config

raph config init
raph config check

Creates local config and data paths under ~/.raph/.

2. Index a repository

raph init --path .
raph init --path . --no-embeddings

Builds the local graph from code, docs, and supported text files.

3. Keep it current

raph sync --path .
raph sync --status

Runs the background worker and reports registered repository state.

4. Clear when needed

raph sync --remove --path .
raph clear --yes

Use explicit cleanup commands when rebuilding local graph state.

MCP and agents

Expose raph to coding agents over stdio.

raph agents mcp setup --path . --dry-run
raph agents mcp setup --path .
raph start

Project-scoped MCP config keeps agent setup close to the repo while the graph remains local.
